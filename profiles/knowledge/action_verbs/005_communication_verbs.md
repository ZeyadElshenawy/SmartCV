# Communication and Influence Action Verbs

Communication verbs describe writing, presenting, teaching, and persuading. They apply across all roles — engineers writing design docs, PMs presenting roadmaps, designers running user research readouts, data scientists publishing findings.

**Writing:**
Authored, Co-authored, Composed, Documented, Drafted, Edited, Outlined, Published, Wrote.

**Speaking and presenting:**
Briefed, Demonstrated, Lectured, Presented, Pitched, Spoke, Trained.

**Persuading and influencing:**
Advocated, Aligned, Championed, Convinced, Influenced, Negotiated, Persuaded, Promoted, Lobbied.

**Teaching and explaining:**
Coached, Demonstrated, Educated, Explained, Facilitated, Guided, Instructed, Mentored, Taught, Tutored, Trained.

**Listening / receiving:**
Consulted, Elicited, Gathered (requirements), Interviewed, Listened, Solicited, Surveyed.

**Translating between audiences:**
Bridged, Communicated, Conveyed, Distilled, Interpreted, Summarized, Synthesized, Translated.

**Verbs to avoid:**
- "Liaised" — corporate jargon; use Worked with, Coordinated, or Represented.
- "Touched base with" — informal cliché.
- "Communicated" by itself — too vague; specify what was communicated and to whom.
- "Talked to" — informal; use Met with, Briefed, Presented to.
- "Wrote up" — informal; use Authored, Documented.

**Pairing communication verbs with outcomes.**
Communication bullets need both an audience scale and a measurable consequence:

- "Authored the engineering hiring rubric used by 12 interviewers across 3 office locations; cut interview-to-decision time from 9 to 4 days."
- "Presented Q3 product roadmap to 80 internal stakeholders; secured sign-off in a single meeting after 2 prior attempts had stalled."
- "Wrote 6 design docs for the migration project, each reviewed by 4+ senior engineers; final architecture shipped with 0 post-launch incidents in the first 90 days."
- "Trained 14 new hires over 3 onboarding cohorts; cohort 3's time-to-first-PR dropped from 8 to 3 days vs. cohort 1."

**Communication-specific scale signals:**
- Audience size: "to N stakeholders", "for N engineers", "across N teams".
- Document distribution: "consumed by N teams", "referenced in N PRs".
- Adoption: "adopted as the standard for N teams".
- Outcome: time saved, decision unblocked, agreement reached, behaviour change.

**Avoid the "demonstrating" closer.** SmartCV's `prompt_guards.py` explicitly bans the pattern `<action>, demonstrating <skill>`. This is especially common in communication bullets ("Presented the roadmap, demonstrating strong communication skills"). Replace with the concrete outcome: "Presented the roadmap; secured sign-off in one meeting after two prior attempts had stalled."

## Concrete rule for SmartCV

Use communication verbs (Authored, Presented, Trained, Wrote) when the source CV documents writing, speaking, teaching, or influencing work. Always include audience size (N stakeholders, N engineers, N hires) and a measurable consequence (decision unblocked, time saved, adoption count). Never end a communication bullet with ", demonstrating <skill>" — replace with the actual concrete outcome.
